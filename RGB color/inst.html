Instructions to play the game

  • Welcome,You are here to play a basic game to find a particular color,on the basis of RGB code.
  • RGB stands for RED,GREEN and BLUE.
  • We have two levels of comforts,easy and hard.Prefering or selecting the mode you can perform the game.
  • if the mode is HARD,six different colors will be displayed among them one color will match to the given respective color code.
  • if the mode is Easy,three different colors will be displayed among them one color will match to the given respective color code.
  • if selecting the wrong color displayed ,the selected color will be disappered.If the right one is selected then all the combinations will turn to the given correct RGB code color.
